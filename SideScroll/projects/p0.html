Rigid Body Simulation

9/16/2013


As a summer project in the past, I set out to create a rigid body simulator. I created a rigid body simulator that could handle convex as well as nonconvex bodies. The simulator used object oriented bounding volume hierarchies for collision detection and a collision resolution model similar to the work in Nonconvex Rigid Bodies with Stacking by Eran Guendelman et al.
